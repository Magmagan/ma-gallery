Ma Gallery is a cross-platform image viewer built with Electron and React. Ma Gallery aims to be simple image browser focused firstly on displaying content.

## Features

* Nice blurred background behind images

## Development

`yarn start`

## Build

`yarn build`

## Testing

`yarn react-test`
